# Sistema de Reportes para Taxis

## Descripción del Proyecto
Este proyecto es un **sistema web de control financiero para taxis**, desarrollado como parte de mi portafolio profesional.  
El sistema digitaliza el control diario de ingresos, gastos y rentabilidad de cada vehículo y conductor.

Permite registrar:
- Ingresos diarios del conductor (por ejemplo, InDriver)
- Gastos de combustible
- Alquiler del vehículo (dividido automáticamente entre propietario y mantenimiento)
- Metas personales y descuentos automáticos (como pensión o ahorro)
- Préstamos entre propietario y conductor

---

## Características Principales
- Autenticación con roles (Administrador, Propietario, Conductor)
- Registro de múltiples vehículos y conductores
- Reportes automáticos y gráficos interactivos
- Exportación a PDF y Excel
- Interfaz responsive e intuitiva
- Historial de acciones

---

## Tecnologías Usadas
**Frontend:** Angular + TailwindCSS + Chart.js  
**Backend:** Spring Boot (Java) + JPA + JWT  
**Base de Datos:** PostgreSQL  
**Control de Versiones:** GitHub (repositorio publico para portafolio)
